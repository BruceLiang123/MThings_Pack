# MThings发布版本包

全新的标准化MODBUS调测工具，提供主从机一体化操作功能集。

支持功能码：01、02、03、04、05、06、15、16；

支持主从机模拟双模式；

支持通信端口包括：串口、TCP客户端、TCP服务器、UDP单播；

支持MODBUS-RTU、MODBUS-ASCII、MODBUS-TCP；

支持在网络通道中嵌入MODBUS-RTU/ASCII;
